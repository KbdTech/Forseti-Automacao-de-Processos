## Descrição

> Como **administrador do sistema**, quero que a plataforma tenha uma tabela de fornecedores vencedores de licitação com API CRUD completa, para que as secretarias possam vincular empresas às ordens de serviço e rastrear gastos mensais por fornecedor.

## Contexto e Motivação

O sistema atualmente não possui nenhum cadastro de fornecedores. A Sprint 11 introduz a fundação do módulo: tabela `fornecedores`, coluna `fornecedor_id` em `ordens` (nullable no banco para preservar histórico, mas **obrigatória no nível da aplicação** para novas ordens), e endpoints CRUD com RBAC. Toda a Sprint 11 (S11.2, S11.3) e Sprint 12 (S12.1–S12.3) dependem desta story.

## Modelo de Dados

### Tabela `fornecedores`

```sql
CREATE TABLE fornecedores (
  id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  razao_social    VARCHAR(255) NOT NULL,
  nome_fantasia   VARCHAR(255),
  cnpj            VARCHAR(14) NOT NULL UNIQUE,          -- 14 dígitos, sem pontuação
  numero_processo VARCHAR(100),                          -- Nº do processo licitatório
  objeto_contrato TEXT,                                  -- Objeto resumido do contrato
  valor_contratado NUMERIC(15, 2),
  data_contrato   DATE,
  banco           VARCHAR(100),
  agencia         VARCHAR(20),
  conta           VARCHAR(30),
  tipo_conta      VARCHAR(20) DEFAULT 'corrente',        -- 'corrente' | 'poupanca'
  secretaria_id   UUID REFERENCES secretarias(id),       -- NULL = global; SET = exclusivo
  is_active       BOOLEAN NOT NULL DEFAULT TRUE,
  created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  updated_at      TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_fornecedores_cnpj        ON fornecedores(cnpj);
CREATE INDEX idx_fornecedores_secretaria  ON fornecedores(secretaria_id);
CREATE INDEX idx_fornecedores_is_active   ON fornecedores(is_active);
```

### Adição em `ordens`

```sql
ALTER TABLE ordens ADD COLUMN fornecedor_id UUID REFERENCES fornecedores(id) ON DELETE SET NULL;
CREATE INDEX idx_ordens_fornecedor_id ON ordens(fornecedor_id);
```

    valor_contratado= Column(Numeric(15, 2), nullable=True)

    banco           = Column(String(100), nullable=True)
    agencia         = Column(String(20), nullable=True)
    conta           = Column(String(30), nullable=True)
    tipo_conta      = Column(String(20), default="corrente")
    secretaria_id   = Column(UUID(as_uuid=True), ForeignKey("secretarias.id"), nullable=True)
    is_active       = Column(Boolean, nullable=False, default=True)
    created_at      = Column(TIMESTAMPTZ, nullable=False, default=func.now())
