## Descrição

> Como **usuário do gabinete, controladoria, tesouraria ou contabilidade**, quero acessar a lista de fornecedores cadastrados no sistema, para verificar quais empresas estão habilitadas para receber pagamentos e comparar com os processos em andamento nas ordens de serviço.

## Contexto e Motivação

A tela de gestão de fornecedores (S11.2) existe apenas para o admin. Os demais perfis operacionais precisam de acesso de leitura à mesma listagem, sem botões de edição/criação/desativação. A secretaria também deve ver os fornecedores, mas apenas os globais + os da sua própria secretaria (scoping já implementado no backend em S11.1).

## Critérios de Aceitação

### Cenário 1: Perfis operacionais acessam /fornecedores

```gherkin
Given que estou autenticado como "gabinete", "controladoria", "contabilidade" ou "tesouraria"
When acesso /fornecedores
Then vejo a tabela de fornecedores com as colunas:
  - Razão Social
  - CNPJ (formatado)
  - Nº Processo
  - Valor Contratado (R$)
  - Secretaria (ou "Global")
  - Status (badge Ativo/Inativo)
And NÃO vejo botões de "Novo Fornecedor", "Editar" ou "Ativar/Desativar"
And filtros de busca e secretaria estão disponíveis (read-only browsing)
```

### Cenário 2: Secretaria acessa /fornecedores com scoping

```gherkin
Given que estou autenticada como "secretaria" da secretaria X
When acesso /fornecedores
Then vejo apenas fornecedores globais (secretaria_id IS NULL)
And fornecedores exclusivos da minha secretaria X
And NÃO vejo fornecedores de outras secretarias
And NÃO vejo botões de edição
```

### Cenário 3: Admin acessa /admin/fornecedores (mantém tela própria)

```gherkin
Given que estou autenticado como "admin"
When acesso /admin/fornecedores (rota do admin)
Then vejo a tela completa com todos os botões de gerenciamento (inalterada)
And o menu lateral exibe "Fornecedores" na seção Admin (com link para /admin/fornecedores)
```

### Cenário 4: Card de fornecedor no OrderDetailModal para todos os perfis

```gherkin
Given que estou autenticado como qualquer perfil
And acesso o detalhe de uma ordem que tem fornecedor vinculado
Then vejo o card "Fornecedor" com razão social, CNPJ, Nº processo e valor contratado
And o card é somente-leitura para todos os perfis
```

### Cenário 5: Filtros disponíveis na tela read-only

```gherkin
Given que estou na tela /fornecedores como perfil operacional
Then vejo campo de busca por nome/CNPJ (debounce 300ms)
And filtro por status (Todos/Ativos/Inativos)
And os filtros funcionam normalmente filtrando os resultados
```

### Cenário 6: Link no menu lateral

```gherkin

Then vejo o link "Fornecedores" no menu lateral
```

## Especificação Técnica

### Rota

**Arquivo:** `frontend/src/routes/index.tsx`

```tsx

{
  path: "/fornecedores",
  element: (
    <RoleGuard roles={["gabinete", "controladoria", "contabilidade", "tesouraria", "secretaria", "admin"]}>
      <FornecedoresPage />
    </RoleGuard>

```

> **Nota:** Admin pode acessar tanto `/fornecedores` (read-only view) quanto `/admin/fornecedores` (full management). O menu lateral do admin pode apontar para `/admin/fornecedores`.
