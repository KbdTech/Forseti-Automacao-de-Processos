# BUG-002 — Layout: Descrição sobrepõe Valor Estimado em Minhas Ordens

**ID:** BUG-002
**Sprint:** S9
**Status:** Done
**Prioridade:** Alta
**Depende de:** US-004, US-021
**Módulo:** Frontend / MinhasOrdensPage

---

## Descrição

> Como **usuário com perfil secretaria**, quero que a tabela de "Minhas Ordens" exiba corretamente todas as colunas sem sobreposição, para que eu consiga visualizar o Valor Estimado sem que o campo de Descrição o encubra.

## Contexto e Motivação

O campo `descrição` da ordem, quando longo, ultrapassa os limites da sua célula e sobrepõe visualmente a célula de `Valor Estimado` ao lado. Isso impede a leitura do valor e compromete a usabilidade da tela de Minhas Ordens.

---

## Critérios de Aceitação

### Cenário 1: Descrição longa é truncada com reticências
```gherkin
Given que acesso "Minhas Ordens" como secretaria
And existe uma ordem com descrição longa (mais de 60 chars)
When a tabela é renderizada
Then o campo Descrição exibe no máximo 2 linhas de texto
And o texto excedente é truncado com "..."
And o Valor Estimado é exibido normalmente na sua coluna
```

### Cenário 2: Tooltip exibe a descrição completa
```gherkin
Given que a descrição está truncada na tabela
When passo o mouse sobre o campo Descrição
Then um tooltip exibe o texto completo da descrição
```

### Cenário 3: Colunas mantêm larguras fixas
```gherkin
Given que a tabela de Minhas Ordens é renderizada
Then a coluna Descrição tem largura máxima definida
And a coluna Valor Estimado não sofre deslocamento
And o layout permanece estável em diferentes tamanhos de tela
```

---

## Especificação Técnica

### Front-End apenas (sem migration, sem alteração de back-end)

**Arquivo principal:**
- `frontend/src/pages/MinhasOrdensPage.tsx`

**Ajuste na célula de Descrição:**
```tsx
// Antes (sem controle de overflow)
<td>{ordem.descricao}</td>

// Depois (com truncate + tooltip)
<td className="max-w-[200px]">
  <span
    className="block truncate"
    title={ordem.descricao}
  >
    {ordem.descricao}
  </span>
</td>
```

**Ajuste na coluna de Valor Estimado:**
```tsx
// Garantir largura mínima para evitar compressão
<th className="w-[120px] min-w-[120px]">Valor Est.</th>
<td className="w-[120px] min-w-[120px] whitespace-nowrap">
  {formatCurrency(ordem.valor_estimado)}
</td>
```

**Configuração da tabela:**
```tsx
// Adicionar table-fixed para controle explícito de colunas
<table className="w-full table-fixed">
```
