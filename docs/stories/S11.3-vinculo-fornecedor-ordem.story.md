## Descrição

> Como **secretaria**, ao criar uma nova ordem de serviço, quero **obrigatoriamente** selecionar o fornecedor vencedor da licitação correspondente, para que o sistema vincule a empresa à ordem e possibilite rastreamento de gastos e pré-preenchimento de dados bancários no pagamento.

## Contexto e Motivação

Toda ordem de serviço, compra ou obra na prefeitura está associada a uma empresa vencedora de licitação. O campo `fornecedor_id` é **obrigatório para todos os tipos de ordem** no nível da aplicação (schema Pydantic + frontend guard). A coluna no banco permanece nullable apenas para preservar ordens históricas existentes sem fornecedor.

A secretaria não pode criar uma ordem sem antes o admin ter cadastrado o fornecedor correspondente (S11.2). O campo Select na Nova Ordem carrega fornecedores ativos da própria secretaria + globais.

**Query de fornecedores:**
```tsx

  queryKey: ['fornecedores-select', debouncedFornecedorSearch],
  queryFn: () => listFornecedores({ q: debouncedFornecedorSearch, is_active: true }),
  staleTime: 1000 * 60,
})
```

**Guard de submit — sempre obrigatório:**
```tsx
// Adicionar ao guard de submit existente (sem condicional por tipo):
const podeSubmeter = /* ... condições existentes ... */ && fornecedorSelecionado !== null
```

**Campo no formulário:**
```tsx
<div className="space-y-2">
  <label className="text-sm font-medium">
    Fornecedor <span className="text-destructive">*</span>
  </label>
  <Select
    value={fornecedorSelecionado?.id ?? ''}
    onValueChange={(id) => {
      const f = fornecedoresData?.items.find(f => f.id === id) ?? null
      setFornecedorSelecionado(f)
    }}
  >
    <SelectTrigger>
      <SelectValue placeholder="Selecionar fornecedor da licitação..." />
    </SelectTrigger>
    <SelectContent>
      <div className="px-2 py-1">
        <Input
          placeholder="Buscar por nome ou CNPJ..."
          value={fornecedorSearch}
          onChange={(e) => setFornecedorSearch(e.target.value)}
          className="h-8 text-sm"
        />
      </div>
      {!fornecedoresData?.items.length && (
        <div className="px-2 py-3 text-xs text-muted-foreground text-center">
          Nenhum fornecedor disponível. Solicite o cadastro ao administrador.
        </div>
      )}
      {fornecedoresData?.items.map(f => (
        <SelectItem key={f.id} value={f.id}>
          {f.razao_social} — {formatCNPJ(f.cnpj)}
        </SelectItem>
      ))}
    </SelectContent>
  </Select>
  {!fornecedorSelecionado && (
    <p className="text-xs text-muted-foreground">
      Selecione o fornecedor vencedor da licitação para esta ordem.
    </p>
  )}
</div>
```
